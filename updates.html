Remarkable Updates and Patches

6/7/2024 - Version 0.0.0

The first prototype release of the game. In this version not much is added but just a few features like the inventory, saving up the game, etc.

16/7/2024 - Version 0.1.0

The biggest update of the game yet. This update took the game from its prototype stage to alpha stage. New features:

  • Now shows version number inside the game.
  • Automatically adjusts texts to make it stay inside the screen.
  • New show FPS setting added.
  • Adjusted alien animations.
  • Adjusted alien spawning time.
  • Made the game a little more mobile-friendly.
  • Fixed a bug in authentication scene.
  • Fixed Terms & Conditions link not being accessible.
  • Fixed bugs where objects won't be saved.
  • More sounds added.
  • Music added.
  • Items are now holdable.
  • Better death.
  • Redesigned crafting UI.

18/7/2024 - Version 0.2.0

This update didn't bring many features but it decreased the development time to finish the game

  • New UI when saving the game.
  • Many more craftable items.
  • Now usable swords.
  • Sword attack animation.
  • New animations.
  • Can now select inventories with numbers.
  • Better alien damage animation.

06/8/2024 - Version 0.4.0

A new update adding new organisms and fixing few bugs. It is one of the biggest update allowing the game to go beta stage faster.

  • Falling meteors.
  • Automatic regeneration.
  • Now able to throw out inventory items.
  • Atmosphere sounds added.
  • Random alien sound during the day added.
  • Giga worm added.
  • Aliens not dealing damage.
